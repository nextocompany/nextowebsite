  <action>
    **Create `src/components/ui/Button.tsx`** (create the directory if needed):

    ```tsx
    import { type ButtonHTMLAttributes } from 'react'

    interface ButtonProps extends ButtonHTMLAttributes<HTMLButtonElement> {
      children: React.ReactNode
    }

    export function Button({ children, className = '', ...props }: ButtonProps) {
      return (
        <button
          className={`
            inline-flex items-center justify-center
            rounded-full
            border border-neutral-200
            bg-white text-neutral-900
            px-6 py-2
            text-sm font-medium
            transition-colors duration-150
            hover:bg-neutral-100
            disabled:opacity-50 disabled:cursor-not-allowed
            ${className}
          `}
          {...props}
        >
          {children}
        </button>
      )
    }
    ```

    CRITICAL: Always pair `border` with `border-neutral-200` explicitly. In TailwindCSS v4, the default border color is `currentColor` (not the light gray of v3). Without the explicit color class, `border` alone on a `text-neutral-900` element produces a dark/black border — wrong for the design.

    Do NOT add `clsx` or `tailwind-merge` — className string concatenation is sufficient for Phase 1 simplicity. These can be added in Phase 2 if needed.

    **Update `src/app/page.tsx`** to demonstrate the complete design system (typography scale + button). This page is replaced by real content in Phase 2 — its purpose here is to make the human verification checkpoint meaningful:

    ```tsx

          <section className="space-y-4">
            <h1>Heading 1 — 40px</h1>
            <h2>Heading 2 — 36px</h2>
            <h3>Heading 3 — 28px</h3>
            <p>Body text at 17px with line-height 1.6. The quick brown fox jumps over the lazy dog.</p>
            <p className="text-neutral-500">Muted body text (neutral-500). Secondary information renders here.</p>
          </section>

          {/* Button demo */}
          <section className="space-y-4">
            <p className="text-neutral-500 text-sm">Button component — outlined, rounded-full:</p>

              <Button>Contact</Button>
              <Button>Products</Button>
            </div>

      )
    }
    ```
